# RichTextEditor

**RichTextEditor** is a WYSIWYG editor built using [Lexical](https://lexical.dev) that is
easy to use and integrate in React projects. It has all the necessary and core features of
an editor and gives you the option to extend it's functionality with your own plugins.

## Quick Start

To start using RichTextEditor, install it first:

<Tabs items={['npm']}>
  <Tabs.Tab>
  ```bash copy
  npm install @poetry-galore/rich-text-editor
  ```
  </Tabs.Tab>
</Tabs>

Use in your project:

```tsx copy
import './App.css'
import RichTextEditor from '@poetry-galore/rich-text-editor';

function App() {
  return (
    <div className="container">
        <RichTextEditor />
    </div>
  );
}

export default App;
```
