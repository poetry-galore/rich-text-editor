# Set Initial Editor State

If you have an already saved editor state and you want to reload
the state in the editor, you can pass the editor state as a string
to the [`initialEditorState`](/docs/props.mdx#initialeditorstate) prop.

- [Load from JSON](#loading-from-a-json-string)
- [Load from HTML](#loading-from-a-html-string)

## Loading from a JSON string

<Steps>

### Example of initial editor state as a JavaScript object:

```js copy filename="initial_editor_state_json.js"
export const InitialEditorState = {
  root: {
    children: [
      {
        children: [
          {
            detail: 0,
            format: 1,
            mode: "normal",
            style: "",
            text: "Be bold and share your thoughts",
            type: "text",
            version: 1,
          },
        ],
        direction: "ltr",
        format: "",
        indent: 0,
        type: "paragraph",
        version: 1,
        textFormat: 1,
      },
    ],
    direction: "ltr",
    format: "",
    indent: 0,
    type: "root",
    version: 1,
  },
};
```

### Import the editor state and stringify it, and pass it to the `initialEditorState` prop.

```tsx {2, 5, 9} copy filename="components/Editor.tsx"
import RichTextEditor from "@poetry-galore/rich-text-editor";
import { InitialEditorState } from "../initial_editor_state_json";

export default function Editor() {
  const initialEditorStateString = JSON.stringify(InitialEditorState);

  return (
    <div className="container">
      <RichTextEditor initialEditorState={initialEditorStateString} />
    </div>
  );
}
```

</Steps>

## Loading from a HTML string

<Steps>

### Example of initial editor state as a JavaScript object:

```ts copy filename="initial_editor_state_html.js"
export const IntialEditorState = `\
<p dir="ltr"><span style="white-space: pre-wrap;">Can I have that </span></p>\
<p dir="ltr"><b><strong class="font-bold" style="white-space: pre-wrap;">Looking at the same place</strong></b></p>\
<p dir="ltr"><i><b><strong class="font-bold italic" style="white-space: pre-wrap;">Can</strong></b></i></p>
`;
```

### Import the editor state and pass it to the `initialEditorState` prop.

```tsx {2, 7} copy filename="components/Editor.tsx"
import RichTextEditor from "@poetry-galore/rich-text-editor";
import { InitialEditorState } from "../initial_editor_state_html";

export default function Editor() {
  return (
    <div className="container">
      <RichTextEditor initialEditorState={InitialEditorState} />
    </div>
  );
}
```

</Steps>
