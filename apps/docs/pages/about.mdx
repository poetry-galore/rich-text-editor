# About RichTextEditor

RichTextEditor was created by [Brian Kimathi](https://github.com/bryokim) for use in `Poetry Galore`, a web app
for writing and posting creative writings. It has now evolved into its own package that can be used in
other upcoming projects that require an editor.

## Team

Currently, the project is maintained by [Brian Kimathi](https://github.com/bryokim). You can check full
list of contributors on [GitHub](https://github.com/poetry-galore/rich-text-editor/graphs/contributors).
